All assignments (1..10) provided as individual .c files in respective folders Assignment_1 .. Assignment_10.
Each file includes only necessary headers:
- Always: stdio.h
- math.h: only where sqrt/fabs used (e.g., triangle area, distance, angle check)
- stdlib.h: only where llabs/abs used
- string.h: only where string functions used

Filenames: Assignment_X/Assignment_X_QY.c where X is assignment number and Y is question number.
